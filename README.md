# CVProjectForTrafficAnalytics
1. Ведется трансляция дорожного движения определенного участка дороги.
2. Данное видео обрабатывается кмпьютерным зрением с целью нахождения автомобилей.
3. Также с помощью компьютерного зрения фиксируется скорость автомобилей и если скорость превышает определенный порог считывается г/н авто.
4. Далее создается какой то список может выпадающий или выплывающий состоящий из г/н авто нарушителей.
5. Затем сделать какой то интерфейс детально еще не продумал например по выбору из списка г/н нарушителя делается запрос в базу 
   где можно было бы получать информацию о владельце авто такую как (ф.и.о., стаж вождения, адрес регистрации, иформация о штрафах оплаченных и не оплаченных и т.д.)
6. Можно реализовать возможность чтобы информация о нарушители отображалась на сайте как определенного авто так и списком сразу нескольких авто.
7. Также можно проверять является ли авто в угоне.
8. Как доп информацию можно выводить на экран об интенсивности и плотности движения (сколько машин проезжает за час, сколько из них грузового транспорта, по каким полосам) необходимую в 
   дальнейшем для предотвращения пробок.
9. Можно попробовать фиксировать нарушения авто которые пересекли сплошную линию или пошли на обгон.Но для этого необходимо обучить сетку дорожной разметке.
10. Все полученные данные можно преобразовать в какие то отчеты, графики либо в автоматические уведомления.
11. Если будет видео с ДТП можно будет сделать фиксацию столкновений для оповещения необходимых служб.
